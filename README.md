# Docker - MySQL - NodeJs 🚀

CRUD de usuarios con NodeJs y MySQL utilizando Docker.

## Instalación 🔧

Se tiene dos ambientes de desarrollo, uno para desarrollo y otro para producción.

### Desarrollo 📦


```
docker-compose -f docker-compose.dev.yml up
```

### Producción🛠️

```
docker-compose up
```
